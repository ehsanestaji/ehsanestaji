### Currently

Building **[PlantGraph](https://plantgraph.se)** — a knowledge-graph platform for plant
genomics and the published literature, focused on *Arabidopsis* and related species,
integrating 100+ data sources.

Adjacent research, with manuscripts in preparation, on resolving-set coordinates for
genome graphs, chromosome-scale pangenome ingest, semantic validation of alignment
formats, and trust-boundary auditing of bioinformatics tools.

### Open source

[**FastaGuard**](https://github.com/ehsanestaji/FastaGuard) — fast, explainable FASTA
QC for assembly files. Rust. The preflight layer that runs before QUAST / BUSCO /
BlobToolKit. Distributed on bioconda: `mamba install -c bioconda fastaguard`.

Most other research code lives in private repositories until the accompanying papers
are public. Preprints and supplementary code releases are linked from
[Google Scholar](https://scholar.google.com/citations?user=nK8EziQAAAAJ&hl=en) and
[ORCID](https://orcid.org/0000-0003-1988-8640) as they appear.
